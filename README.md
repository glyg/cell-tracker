# cell-tracker

adaptation of bnoi/scikit-tracker to track cells

## Depencies

### Required

* numpy >= 1.8
* scipy >= 0.12
* pandas >= 0.13
* scikit-image >= 0.9
* scikit-learn >= 0.13
* matplotlib >= 1.8
* sktracker >= 0.2-dev


### Optionnal (for UI elements)

* ipython >= 1.0
* pyqt4 >= 4.10
